Turtle Core I

Voir sur GitHub
Logo du projet

Ce projet vise à concevoir, simuler et émuler une architecture de CPU utilisant un bus 8 bits. L'objectif est d'en apprendre plus sur le fonctionnement interne d'un CPU et la programmation bas niveau. Les détails techniques du projet sont disponibles sur la page GitHub avec le code source.

Diagramme d'architecture

La simulation du comportement du processeur est faite avec Logisim Evolution, un logiciel qui permet de créer des circuits logiques et de les simuler.

Capture d'écran de la page Logisim

Pour faciliter la programmation des PLA, dans le bloc logique décodant les opérations, les différents cycles sont définissables en python pour une meilleure lisibilité et une plus grande flexibilité.
